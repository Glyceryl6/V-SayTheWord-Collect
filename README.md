# V圈哦齁/拟声词挑战视频收集
本项目主要用于收集在25年末以及26年初在B站等平台上出现的各种 **哦齁/本子拟声词挑战** 视频，大家可以在这里分享一些自己看过并且缓存过但是在项目中没收集到的类似视频。

**PS：本子拟声词挑战一开始是从X上面传进来的，这里可以不用收集X上面的（你都有上X的能力了，应该也不会只看这种视频了罢[心虚]）**

## 提交说明

### ⚙️使用方法

  - **开发者：** 可以先把项目Fork或者Clone下来，然后新建一个issue进行提交即可，如果Clone下来比较耗时，也可以直接提交PR
### 📃文件名相关
**PS：并不是强制要求，只是为了便于大家得知是哪位UP主的作品。**

1. **最好的情况：** 未被下架或换源，则使用原视频标题，前面带上原UP主的名字，例如“【UP主名字】视频标题”；
2. **较好的情况：** 已换源或下架，但是有原视频的缓存，则方法同上（缓存是可以看到原视频的标题的）；
3. **较差的情况：** 已换源或下架，并且没有原视频的缓存，而是从其他地方下载来的，然后并不知道该视频的原标题，如果有UP主的水印，或者知道是哪位UP主，就只保留该UP主的名字即可。有些视频下架之后，部分UP主可能会发动态吐槽，可能会看到视频的原标题（虽然没必要这么认真，但这里只是说明一下可行性）；
4. **最差的情况：** 在上述条件均不满足、甚至连原作者都不知道是谁的情况下，那当前的文件名是什么就用什么。

### 🎁内容相关
1. 有原视频优先提交原视频，尽量以原作者发布的第一版为主。在画质相同或者差别不大的情况下，也可以优先使用MP4格式或者体积较小的视频；
2. 如果原视频已被下架、换源或者补档，有缓存视频则提交缓存视频，如果只提供了音频，也可以提交音频（如果还有换源视频，也可以用剪辑工具将它们合并在一起）；
3. 如果没有缓存视频，或者实在找不到更好的，但是有录屏视频，那也可以；
4. 部分UP主可能会上传经过处理的视频（比如前后加入很长的无意义内容、倒放、声音调低等），建议先使用剪辑工具恢复成“原貌”之后再进行上传；
5. 不要上传使用AI制作或者使用直播时发出的“怪动静”等剪辑而成的视频；
6. **优先以UP主直接发布出来的为主（包括但不限于公开的视频或网盘），如果是因为平台审核导致下架而转为加群或者其它方式获取的，可以根据实际情况选择是否上传；**
7. **有的UP主本身就没打算做这类视频，或者只是发布了恶搞/棒读视频，则不要为了上传而要求该UP主去做这类视频，更不要使用AI去做该UP主的这类视频。**

## 💦常见问题说明
- ### 我缓存了视频，但是怎么把视频给弄出来？
以安卓手机为例：
  1. 打开你手机的“文件管理”，找到“Android”文件夹（旁边会显示“安卓”二字）；
  2. 点进去会发现有个旁边带有“安卓数据”的data文件夹，点进去的话会提示“无法访问文件”；
  3. 然后根据提示前往“安卓储存访问框架”查看文件，然后再次找到这个文件夹；
  4. 打开data文件夹之后，往下拉，找到一个名为“tv.danmaku.bili”的文件夹；
  5. 打开之后，里面的download文件夹就是缓存的文件，只不过这个是音视频分离的；
  6. 如果不知道怎么还原成原视频，可以把这些文件压缩一下打包发出来，我自己来处理。
